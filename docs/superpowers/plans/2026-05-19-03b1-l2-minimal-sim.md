# L2 Minimal Vertical Slice (03b-1) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement relaxed L2 logic in `consensus` plus a deterministic `sim` driver so `happy_path` produces real `BroadcastMicroQc` traffic and checkers pass.

**Architecture:** `StateMachine::step(event, &HostContext)` mutates an in-memory `Book` and returns `Action`s. `apps/sim` owns vertex production, `VirtualNet` delivery, `store_micro_qc` on broadcast, and checkers via `Persistence::micro_qc_for`. `node`/`cli` compile with stub or minimal context.

**Tech Stack:** Rust 1.88, `consensus`, `types`, `crypto`, `sim`, `storage`, `borsh`, `smallvec`.

**Spec:** `docs/superpowers/specs/2026-05-19-l2-sim-milestone-a-design.md`

Logic (spec §5.2):

```rust
pub fn on_certified_vertex(
    book: &mut Book,
    cv: CertifiedVertex,
    ctx: &HostContext<'_>,
) -> Result<Actions> {
    let hash = cv.vertex.hash;
    book.seen.insert(hash, (cv.vertex.round, cv.vertex.author));
    let w = WaveId::of_round(cv.vertex.round).0;
    if !wave_ready_to_commit(w, ctx.dag)? {
        return Ok(Actions::new());
    }
    // Linearize the wave (relaxed: sort by (round, author) per spec §5.2)
    // then derive the checkpoint hash before we check / record emission.
    let linearized = linearize_wave_sorted(w, ctx.dag)?;
    let checkpoint_hash = checkpoint_hash_from_rounds(&linearized);
    if book.emitted_micro_qc.contains(&checkpoint_hash) {
        return Ok(Actions::new());
    }
    let qc = build_flat_micro_qc(w, checkpoint_hash, ctx)?;
    book.emitted_micro_qc.insert(checkpoint_hash);
    book.wave_status.insert(w, WaveStatus::Committed);
    Ok(smallvec![Action::BroadcastMicroQc(qc)])
}
```

`wave_ready_to_commit`: anchor round `4w+3` has vertex in dag; each round `4w..=4w+3` has `vertices_at_round` non-empty.

`build_flat_micro_qc`: bitmap all validators that have any vertex in wave (equal stake → count ≥ `2f+1` when factory filled all rounds).

```rust
pub fn enqueue_from_action(
    &mut self,
    sender: u32,
    action: &Action,
    validator_count: u32,
    now: u64,
    _rng: &mut ChaCha20Rng,
) {
    if let Action::BroadcastMicroQc(qc) = action {

            }
            self.enqueue(InFlight {

                deliver_at: now,
            });
        }
    }
```

```rust
pub fn build_certified_vertex(
    virtual_round: u64,
    proposer_index: u32,
    parent_hash: Option<Hash32>,
) -> CertifiedVertex {
    let author = validator_id_for_index(proposer_index);
    let hash = vertex_hash(virtual_round, &author);
    CertifiedVertex {
        vertex: Vertex {
            round: Round(virtual_round),
            author,
            parents: parent_hash.into_iter().collect(),
            blobs: vec![],
            hash,
        },
        certificate: fixture_certificate(),
    }
}

fn vertex_hash(round: u64, author: &ValidatorId) -> Hash32 {
    let mut buf = Vec::with_capacity(8 + 32);
    buf.extend_from_slice(&round.to_be_bytes());
    buf.extend_from_slice(&author.0);
    let bytes = blake3_with_dst(dst::SIM_VERTEX_HASH, &buf);
    Hash32(bytes)
}
```

- [ ] **Step 2: Implement `apply_actions`**

```rust
fn apply_actions(
    world: &mut World,
    validator_idx: u32,
    actions: consensus::Actions,
    now: u64,
) {
    for action in actions {
        match action {
            Action::BroadcastMicroQc(qc) => {
                world.persistence[validator_idx as usize]
                    .store_micro_qc(&qc)
                    .unwrap();
                world.net.enqueue_from_action(
                    validator_idx,
                    &Action::BroadcastMicroQc(qc),
                    world.machines.len() as u32,
                    now,
                    &mut world.rng,
                );
            }
            Action::ScheduleTimer { id, delay_nanos } => {
                world.timer.schedule(now + delay_nanos as u64, id);
            }
            Action::CancelTimer(id) => world.timer.cancel(id),
            Action::PersistMacroQc(qc) => {
                // Macro path lands in 03c; keep the port warm so the sim
                // can already exercise persistence wiring.
                let _ = world.persistence[validator_idx as usize].store_macro_qc(&qc);
            }
            Action::UpdateBlobStatus { .. } => {
                // Optional in 03b-1: trace only. Plan 03c surfaces this.
            }
            Action::BroadcastMacroProposal(_)
            | Action::BroadcastBlsPartial(_)
            | Action::BroadcastSubnetAggregate(_)
            | Action::BroadcastMacroQc(_)
            | Action::EmitSlashEvidence { .. } => {
                // 03b-1: honest L2-only flow never emits these. Debug-assert
                // in tests, log in release so unexpected emissions surface.
                debug_assert!(false, "unexpected non-L2 action in 03b-1: {action:?}");
            }
        }
    }
}
```
